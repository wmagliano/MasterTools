\# MasterTools



Android frontend for Kali NetHunter tools.



\## Features

\- Run tools inside Kali NetHunter

\- Nmap

\- WhatWeb

\- TestSSL

\- MasterTools script



\## Requirements

\- Root access

\- Kali NetHunter installed

\- `/data/local/nhsystem/nh\_run.sh`

\- `/root/mastertools.sh`
